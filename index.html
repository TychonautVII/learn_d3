Description

I used a simple, open, data set to apply machine learning to the problem of identifying heart disease from a series of physiological measurements. I assessed the performance of Perceptrons, Gaussian Mixtures, and Support Vector Classifiers.

In total 3081 Models were trained, 25 times each for cross validation. Statistics on performance of each of these 3081 models across the 25 cross validation runs are displayed above.
